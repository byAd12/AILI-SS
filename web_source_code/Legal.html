Forma de usar Nuestro Logo

Para utilizar nuestro logo desbes cumplir una serie de reglas:


Usarlo sobre un fondo que no contraste con el logo.
No modificar el logo con filtros.
Respetar el tamaño de la imágen no haciéndolo muy pequeño.
No editar la imágen.
