En esta página podrás ver las novedades de cada versión desde que se lanzó el software.

Versión 2.0

x/x/2025

Descargar (Portable) Descargar (Instalador) Reportar un Error

Nueva interfaz gráfica más intuitiva y limpia.


Nuevo servidor de claves hospedado en render.com.


Nuevas funcionalidades de criptografía, herramientas de red, mejora de funcionalidades, arreglo de funcionalidades existentes.


Bloqueo de funcionalidades según si aceptas los términos.


Mejor personalización y posibilidad de hacer copias de seguridad.


Nuevo blog más completo y con páginas.


Shortcuts, Nuevas herramientas, posibilidad de exportar resultados, y mucho más.

Versión 1.6

24/12/2024

Descargar (Windows) Reportar un Error

Ahora se puede customizar el color de los títulos, del texto normal y del fondo de cada ventana. Para ello vaya a "Config.json".

Se arreglaron pequeños errores.

Versión 1.5

18/12/2024

Descargar (Windows) Reportar un Error

Se añadió un nuevo botón en la página principal "Descargar Dependencias", donde detecta que si tienes Python3 y te instala las librerias necesarias para todas las funciones, si no te lleva a la página principal de descarga de Python3 y te avisa que tienes que hacer al instalarlo.


Se añadió una descripción más detallada de algunas ventanas para ser más intuitivo.


Se arreglaron pequeños errores.

Versión 1.4

16/12/2024

Descargar (Windows) Reportar un Error

Ahora, en el botón "Config.json", se detecta automáticamente la dirección de red en la sugenda entrada de texto para ahorrar trabajo.


Se mejoró la página web del botón "Interfaces e IPs" con animaciones y otro formato.


Se renombro el botón "Detección en Red" a "Monitorización en Red".


Se añadieron más mensajes aleatorios entre ventanas.


Se añadió más información en "Información Técnica".

Versión 1.3

14/12/2024

Descargar (Windows) Reportar un Error

Se arreglaron los enlaces de la página información y además se cambió su estética.


Se quitaron los scrollbar innecesarios en las ventanas pequeñas para que se vean más limpias.


En cada pestaña principal se añadió un pequeño texto con el dominio de la página web principal.


Se añadió un nuevo botón en Herramientas de red: "Interfaces e IPs", donde se abrirá un web server para ver interfaces, sus IPv4 relacionados y su estado.

Versión 1.2

13/12/2024

Descargar (Windows) Reportar un Error

Se añadió un fichero al .zip con información sobre la aplicación.


Se cambió el formato de las fechas de las claves.


Se añadió un recordatorio en el botón "Buscar IP por MAC" para ejecutarse en sudo o Administrador.


Se arreglaron los botones de la pestaña "Detección en Red".
